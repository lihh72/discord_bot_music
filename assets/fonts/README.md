# Fonts Directory

Place `NotoSans-Regular.ttf` in this directory for cross-platform lyric rendering.

## Download

Download from Google Fonts:
https://fonts.google.com/noto/specimen/Noto+Sans

1. Click "Download family"
2. Extract the ZIP
3. Copy `NotoSans-Regular.ttf` to this directory

If the font file is not present, the VideoGenerator will fall back to Pillow's
default bitmap font (which has limited size/quality support).
